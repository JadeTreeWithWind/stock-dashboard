## <a name="introduction">✨ 專案介紹</a>

這是一個 **現代化的股票市場儀表板**，使用 **React、Vite、TypeScript 與 Tailwind CSS** 打造。

專案致力於提供「**Premium 級別的視覺體驗**」，採用深色模式與 **Glassmorphism (玻璃擬態)** 設計風格。整合了豐富的 **TradingView Widgets**，為投資者提供從宏觀市場概覽、熱力圖到個股深度技術分析的全方位資訊。

整體架構採用 **Component-based 設計**，確保程式碼的可維護性與擴充性，並針對各類裝置進行了響應式優化，無論是桌面看盤還是行動裝置查詢，都能享有流暢的操作體驗。

## <a name="tech-stack">⚙️ 技術棧</a>

- **[React](https://reactjs.org/)**
  前端核心框架，利用 Hook 與 Component 機制建構高效的使用者介面，確保狀態管理與畫面渲染的效能。

- **[Vite](https://vitejs.dev/)**
  新一代前端建構工具，提供極速的開發伺服器啟動與熱模組替換 (HMR)，大幅提升開發效率。

- **[TypeScript](https://www.typescriptlang.org/)**
  提供強型別支援，在開發階段即能捕捉潛在錯誤，提升程式碼的健壯性與 IntelliSense 體驗。

- **[Tailwind CSS](https://tailwindcss.com/)**
  Utility-first CSS 框架，搭配 `tailwindcss-animate` 與自定義配置，快速實現複雜的深色玻璃擬態風格與排版。

- **[Radix UI](https://www.radix-ui.com/)**
  無樣式 (Headless) UI 組件庫，作為 Dialog、Popover 等互動元件的基礎，確保無障礙性 (Accessibility) 與功能完整性。

- **[TradingView Widgets](https://www.tradingview.com/widget/)**
  整合多種專業金融圖表與數據模組，包含先進燭光圖、市場熱圖、財報分析等，提供專業級的看盤功能。

## <a name="features">🔋 功能特色</a>

👉 **全球市場脈動 (Market Overview)**
首頁即時展示全球主要指數行情，讓您一眼掌握市場多空趨勢。

👉 **股票熱力圖 (Stock Heatmap)**
以視覺化色塊呈現各板塊漲跌幅，快速識別市場熱點與資金流向。

👉 **個股深度分析 (Stock Detail)**
點擊或搜尋個股代碼，即可查看包含 K 線圖、技術指標、公司簡介與財務報表的詳細數據。

👉 **即時新聞串流 (News Feed)**
整合股市新聞時間軸，不錯過任何影響股價的重大消息。

👉 **智慧搜尋功能 (Smart Search)**
支援快速搜尋股票代碼，流暢切換至個股詳情頁面。

👉 **精緻玻璃擬態 UI (Premium Glassmorphism)**
全站採用深色玻璃質感設計，搭配細膩的微動畫 (Micro-interactions) 與漸層效果，提供沉浸式的視覺享受。

👉 **完整響應式設計 (Responsive Design)**
針對 Desktop、Tablet 與 Mobile 螢幕優化排版，隨時隨地輕鬆掌握投資動態。
